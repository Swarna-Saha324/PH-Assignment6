# 🛒 DigiTools - Premium Digital Products Marketplace

DigiTools is a modern, high-performance web application designed for browsing and managing premium digital tools and templates. It delivers a seamless user experience with a responsive design, dynamic cart management, and an interactive UI.

## ⚙️ Tech Stack

- ⚛️ React.js  
- ⚡ Vite  
- 🎨 Tailwind CSS  
- 🌼 DaisyUI  
- 🎯 Lucide React  
- ⭐ Font Awesome  
- 🔔 React-Toastify  

---

## ✨ Features

- 🛍️ **Curated Collection**
  - Browse premium AI tools and design templates

- 🛒 **Dynamic Shopping Cart**
  - Add/remove items instantly
  - Automatic total price calculation

- 🎯 **Interactive UI/UX**
  - Clean and modern design
  - Smooth user experience

- 📱 **Responsive Navbar**
  - Fully responsive across devices
  - Dynamic cart badge updates in real-time

- 🔔 **Toast Notifications**
  - Instant feedback for user actions
  - Example: "Item deleted!"

3.Install dependencies:

4.Start the development server:
npm run dev
5.Open in browser:
http://localhost:517
